# Sistem Kalkulator Web Responsif (Judul 5 Praktikum Pemrograman Web)

Proyek ini merupakan hasil implementasi **Tugas Akhir/Praktikum Pemrograman Web (Judul 5)**. Fokus utama proyek adalah membangun kalkulator *desktop-style* dengan antarmuka web modern yang memiliki fungsionalitas matematika yang canggih dan kemampuan interaksi yang komprehensif.

---

## Tujuan dan Fitur Utama

Proyek ini mendemonstrasikan penguasaan terhadap *front-end development* dan logika perhitungan kompleks.

* **Logika BODMAS/PEMDAS:** Perhitungan selalu mengikuti urutan operasi matematika yang benar (Perkalian dan Pembagian didahulukan, diikuti Penambahan dan Pengurangan).
* **Responsif Penuh:** Mendukung interaksi yang mulus melalui **klik layar/sentuh** dan **input *keyboard***.
* **Kontrol Clear Dinamis (AC/C):** Tombol *Clear All* (AC) secara otomatis bertransisi menjadi **C** (*Clear Entry*) setelah input pertama.
* **Desain Dark Mode Profesional:** Antarmuka bersih, dirancang dengan efek *shadow* (bayangan) pada kontainer utama untuk elevasi visual.

---

## Fitur Data dan Memori Canggih

Kalkulator ini dilengkapi dengan panel tersembunyi untuk manajemen sesi perhitungan:

### Riwayat (History)
* Mencatat dan menampilkan semua ekspresi dan hasil perhitungan.
* Pengguna dapat mengklik hasil di riwayat untuk memuat kembali angka tersebut ke layar utama.

### Memori (M)
* **M+ / M-**: Menambah atau mengurangi nilai tampilan saat ini dari variabel memori.
* **MR (Memory Recall)**: Mengambil dan menampilkan nilai memori.
* **MC (Memory Clear)**: Mengatur ulang nilai memori ke nol.

##  Panduan Penggunaan Proyek

Proyek ini hanya membutuhkan *browser* modern untuk dijalankan.

### 1. Menjalankan Aplikasi
Buka file **`index.html`** langsung di *browser* Anda. Tidak diperlukan *server* lokal atau instalasi *dependency*.

### 2. Peta Tombol Keyboard
Untuk interaksi *desktop* yang optimal:
* **Angka & Operator:** Gunakan tombol `0-9`, `+`, `-`, `*` (Perkalian), dan `/` (Pembagian).
* **Hitung Hasil:** Tekan `Enter` atau `=`.
* **Clear Utama:** Tekan tombol `Esc` (Escape) untuk memicu fungsi AC/C.
* **Hapus Digit:** Gunakan tombol `Backspace`.
